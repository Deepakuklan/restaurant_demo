# TANGO Restaurant POS

Modern point-of-sale system for restaurants.

## Quick Start

```bash
flutter run
```

**Login:** 1234

## Features

- 5 sections, 44 tables
- 6 menu categories
- Product customization (cooking, sides, sauces)
- Automatic tax calculation (18% ITBIS + 10% LEY)
- Works offline with SQLite
